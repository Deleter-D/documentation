| 输入    | 编号 | 有效等价类  | 编号 | 无效等价类        |
| ------- | ---- | ----------- | ---- | ----------------- |
| 图片URL | 1    | URL格式正确 | 2    | URL协议不正确     |
|         |      |             | 3    | URL路径不正确     |
|         |      |             | 4    | URL图片UUID不正确 |
|         |      |             | 5    | URL格式不正确     |
